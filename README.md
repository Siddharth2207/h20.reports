# Raindex OrderBook Depth Chart
## dev
To start, install deps by following command:
```sh
nix develop -c install-deps
```
or enter nix shell first
```sh
nix develop
```
and then run:
```sh
install-deps
```
---
To run the website:
```sh
nix develop -c npm run start
```
or enter nix shell first
```sh
nix develop
```
and then run:
```sh
npm run start
```
---
